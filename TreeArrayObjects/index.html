Fall is Coming!

I modified the "TreeArrayObjects" example to include:

  • Branch widths calculated using a decaying exponential model based on branch generation.
  • Branch angles and lengths are set within various random ranges to give a more realistic tree-like structure.
  • Leaves which change from green to a random color between yellow and red.
  • When a leaf reaches its final color it falls to the ground.
  • As the leaf falls it is affected by a Perlin noise wind.
  • The leaves collect on the ground.
